# Instaliacija

``cd /root/
``

``
wget https://raw.githubusercontent.com/lukasenka/rehlds-installer/main/rehlds.sh
``

``
chmod +x rehlds.sh
``

``
./rehlds.sh
``

Gero naudojimo :)
